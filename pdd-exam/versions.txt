27/09/2016 version 1.0

Закончен переход на БД SQLite для хранения вопросов.
1. Вопросы хранятся в БД SQLite
2. Иллюстрации к билетам хранятся в каталоге (без архива)
3. Названия иллюстарций формируются по принципу <№темы>-<№вопроса в теме>.jpg
4. В БД есть таблица соответствия вопросов по темам и вопросов по билетам. (Билеты были опубликованы на официальном интернет ресурсе ГИБДД).
5. На основе билетов из вопросов сформированы тематические блоки, которые используются при формировании билетов в программе.
